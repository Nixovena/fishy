installation

installation is handled via the debian installer for maximum compatibility and minimal overhead. boot from the live media and select the install option from the grub menu.

iso verification

before flashing, ensure the iso is not corrupted. verify the sha256 checksum.

sha256sum fishy-linux.iso

compare the output with the hash provided on the release page.

custom commands

fishy ship minimal wrappers to make your life easier in the terminal.

system update

updates the system packages, resolving dependencies quietly.

sudo update

system info

displays system resources and distribution info using the fishyfetch script.

fishyfetch

flatpak setup

installs flatpak, adds the flathub repository, and sets up integrating components.

flatpak-setup

zram management

easy zram installation, configuration and management script.

zram-settings

cpu mitigations

enables or disables cpu security mitigations (spectre/meltdown) based on your hardware year and model.

mitigations-settings

developed by nixovena labs.

debian-based. free.
